# AI Optimization Simulator

## Deskripsi
Aplikasi simulasi algoritma optimasi yang membandingkan:

- Hill Climbing
- Simulated Annealing
- Genetic Algorithm

pada fungsi Rastrigin.

## Fitur
- Visualisasi grafik konvergensi
- Pengaturan parameter algoritma
- Perbandingan performa algoritma
- Antarmuka berbasis web

## Teknologi
- HTML
- CSS
- JavaScript
- Chart.js

## Cara Menjalankan
1. Download repository
2. Buka file index.html
3. Jalankan simulasi
